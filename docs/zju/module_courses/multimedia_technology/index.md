### 考核方式

1. 小测：20%
2. project：40%

    1. project 成果：30%
    2. project 展示：10%

3. 期末考试：40%
4. Bonus（论文阅读）：5%

#### 小测

小测开卷，几乎每节课结束前都会有小测，小测的内容一直没变过，我在 [个人笔记](#个人笔记) 当中也有所整理

#### project

选题和这门课学的东西有关即可，不清楚自己的选题合不合适可以先去找老师问一问。例如：文本压缩、图像压缩、视频压缩等

project 可以多人组队，但是小组成员赋分的规则是根据贡献度分配的，例如：`1 : 0.9 : 0.8 : 0.7`，**不允许 `1 : 1 : 1 : 1`，即每个人分数相同的分配方法**。因此，这门课的 project 是鼓励单人完成的

#### 期末考试

期末考试开卷，可携带任何纸质资料
