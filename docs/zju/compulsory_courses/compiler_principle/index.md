### 考核方式

1. 课程作业：10%
2. quiz：10%
3. lab：30%。共 5 次
4. 期末考试：50%

quiz 学在浙大提交，截止时间是当周周天

期末考试可以携带 3 张 A4 纸

lab 实验作业有很多 bonus 可以写
