#### 期末考试

##### 考试题型

1. 程序输出题 30 分，6~8 道
2. 选择题 20 分，10 道
3. 填空题 30 分，2 道大填空题
4. 函数 / 类设计题 20 分

##### 考试内容

1. class, constructor and destructor, inheritance, polymorphism, virtual
2. pass by reference, static member variable
3. template, exception
4. operator overloading, casting
