# 常用软件

你可以找到一些软件的使用教程或小技巧

**Adobe Bridge** 

管理各种图片、视频、音频等素材

## Lr

**Adobe Lightroom**

管理和编辑图片，图片调色

## Ps

**Adobe Photoshop**

图像编辑、处理和图形设计

## Pr

**Adobe Premiere Pro**

视频编辑

## Ae

**Adobe After Effects**

动态图形和视觉效果合成

## Au

**Adobe Audition**

音频编辑和处理

## Me

**Adobe Media Encoder**

视频和音频编码
