# Ae 给物体添加阴影

教程视频：[这个插件越早知道的越好 VFX特效合成插件——shadow投影阴影影子](https://www.bilibili.com/video/BV1Pt4y1Q7vf/){:target="_blank"}

!!! info "注意"

    需使用 VFX Suite 红巨人特效插件

## 添加效果

RG VFX -> Shadow

## 基本流程

1. 设置阴影的平面轴
2. 调整阴影方向
3. 调节具体参数

## 参数

### 阴影平面

设置阴影平面轴

### 阴影样式

设置阴影样式

1. 阴影颜色
2. 透明
3. 柔软
4. 淡化
5. 缩放
6. 偏移

### 阴影弯曲

实现阴影弯曲效果，比如影子遇到墙会弯曲

### 阴影混合

1. 伽马
2. 亮度
3. 淡化噪波
4. 后期淡化

### 置换

利用贴图更改阴影纹理
