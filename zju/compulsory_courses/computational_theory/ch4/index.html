    1 The Definition of A Turing Machine¶

    жң¬иҙЁдёҠпјҢдёҖеҸ° Turing machines з”ұдёҖдёӘжңүйҷҗзҠ¶жҖҒжҺ§еҲ¶еҚ•е…ғе’ҢдёҖжқЎзәёеёҰз»„жҲҗгҖӮдёӨиҖ…д№Ӣй—ҙзҡ„йҖҡдҝЎз”ұдёҖдёӘеҚ•зӢ¬зҡ„зЈҒеӨҙжҸҗдҫӣпјҢиҜҘзЈҒеӨҙд»ҺзәёеёҰдёҠиҜ»еҸ–з¬ҰеҸ·пјҢд№ҹз”ЁдәҺжӣҙж”№зәёеёҰдёҠзҡ„з¬ҰеҸ·гҖӮжҺ§еҲ¶еҚ•е…ғд»ҘзҰ»ж•Јзҡ„жӯҘйӘӨиҝҗиЎҢпјӣеңЁжҜҸдёҖжӯҘпјҢе®ғж №жҚ®е…¶еҪ“еүҚзҠ¶жҖҒе’ҢиҜ»/еҶҷзЈҒеӨҙеҪ“еүҚжү«жҸҸеҲ°зҡ„зәёеёҰз¬ҰеҸ·жқҘжү§иЎҢдёӨдёӘеҠҹиғҪпјҡ

    1. е°ҶжҺ§еҲ¶еҚ•е…ғзҪ®дәҺдёҖдёӘж–°зҠ¶жҖҒ

    2. жү§иЎҢд»ҘдёӢж“ҚдҪңд№ӢдёҖпјҡ

      1. еңЁеҪ“еүҚжү«жҸҸзҡ„зәёеёҰж–№ж јдёӯеҶҷе…ҘдёҖдёӘз¬ҰеҸ·пјҢжӣҝжҚўжҺүе·Іжңүзҡ„з¬ҰеҸ·
      2. е°ҶиҜ»/еҶҷзЈҒеӨҙеҗ‘е·ҰжҲ–еҗ‘еҸіз§»еҠЁдёҖдёӘзәёеёҰж–№ж ј

    зәёеёҰжңүдёҖдёӘе·Ұз«ҜпјҢдҪҶеҗ‘еҸіж— йҷҗе»¶дјёгҖӮдёәдәҶйҳІжӯўжңәеҷЁе°Ҷе…¶зЈҒеӨҙз§»еҮәзәёеёҰзҡ„е·Ұз«ҜпјҢжҲ‘д»¬еҒҮи®ҫзәёеёҰзҡ„жңҖе·Ұз«ҜжҖ»жҳҜз”ұдёҖдёӘзү№ж®Ҡз¬ҰеҸ· \(\rhd\) ж Үи®°пјӣжҲ‘д»¬иҝӣдёҖжӯҘеҒҮи®ҫпјҢжҲ‘д»¬жүҖжңүзҡ„еӣҫзҒөжңәйғҪжҳҜиҝҷж ·и®ҫи®Ўзҡ„пјҡжҜҸеҪ“зЈҒеӨҙиҜ»еҲ° \(\rhd\) ж—¶пјҢе®ғз«ӢеҚіеҗ‘еҸіз§»еҠЁгҖӮжӯӨеӨ–пјҢжҲ‘д»¬е°ҶдҪҝз”ЁдёҚеҗҢзҡ„з¬ҰеҸ· вҶҗ е’Ң вҶ’ еҲҶеҲ«иЎЁзӨәзЈҒеӨҙеҗ‘е·ҰжҲ–еҗ‘еҸіз§»еҠЁпјӣжҲ‘д»¬еҒҮи®ҫиҝҷдёӨдёӘз¬ҰеҸ·дёҚеұһдәҺжҲ‘д»¬иҖғиҷ‘зҡ„д»»дҪ•еӯ—жҜҚиЎЁ

    йҖҡиҝҮе°Ҷиҫ“е…Ҙеӯ—з¬ҰдёІеҲ»еҶҷеңЁзәёеёҰе·Ұз«ҜгҖҒзҙ§йӮ» \(\rhd\) з¬ҰеҸ·еҸідҫ§зҡ„зәёеёҰж–№ж јдёҠпјҢдёәеӣҫзҒөжңәжҸҗдҫӣиҫ“е…ҘгҖӮзәёеёҰзҡ„е…¶дҪҷйғЁеҲҶжңҖеҲқеҢ…еҗ« blank з¬ҰеҸ·пјҢи®°дёә \(\sqcup\)гҖӮжңәеҷЁеҸҜд»ҘжҢүе…¶и®ӨдёәеҗҲйҖӮзҡ„д»»дҪ•ж–№ејҸиҮӘз”ұж”№еҸҳе…¶иҫ“е…ҘпјҢе№¶еҸҜд»ҘеңЁеҸідҫ§ж— йҷҗзҡ„з©әзҷҪзәёеёҰйғЁеҲҶиҝӣиЎҢеҶҷе…ҘгҖӮз”ұдәҺжңәеҷЁдёҖж¬ЎеҸӘиғҪе°Ҷе…¶зЈҒеӨҙз§»еҠЁдёҖдёӘж–№ж јпјҢеңЁд»»дҪ•жңүйҷҗж¬Ўзҡ„и®Ўз®—д№ӢеҗҺпјҢеҸӘжңүжңүйҷҗеӨҡдёӘзәёеёҰж–№ж јдјҡиў«и®ҝй—®еҲ°

    A Turing machine is a quintuple \((K,\Sigma,\delta,s,H)\), where

    1. \(K\) is a finite set of states
    2. \(\Sigma\) is an alphabet, containing the blank symbol \(\sqcup\) and the left end symbol \(\rhd\), but not containing the symbols \(\leftarrow\) and \(\rightarrow\)
    3. \(s \in K\) is the initial state
    4. \(H \subseteq K\) is the set of halting states

    5. \(\delta\), the transition function, is a function from \((K-H)\times \Sigma\) to \(K \times (\Sigma \cup \lbrace\leftarrow, \rightarrow \rbrace)\) such that

      1. for all \(q \in K - H\), if \(\delta(q,\rhd) = (p,b)\), then \(b = \rightarrow\)
      2. for all \(q \in K - H\) and \(a \in \Sigma\), if \(\delta(q,a) = (p,b)\), then \(b \not ={ \rhd}\)

    A configuration of a Turing machine \(M = (K,\Sigma,\delta,s,H)\) is a member of \(K \times \rhd \Sigma^* \times (\Sigma^*(\Sigma - \lbrace \sqcup \rbrace) \cup \lbrace e\rbrace)\)

    1. з¬¬дёҖдёӘдҪҚзҪ®пјҡиЎЁзӨәеӣҫзҒөжңәеҪ“еүҚзҡ„зҠ¶жҖҒ
    2. з¬¬дәҢдёӘдҪҚзҪ®пјҡиЎЁзӨәд»ҺзәёеёҰе·Ұз«ҜејҖе§ӢпјҢеҲ°еҪ“еүҚзЈҒеӨҙжүҖеңЁдҪҚзҪ®пјҲеҢ…еҗ«иҜҘдҪҚзҪ®з¬ҰеҸ·пјүдёәжӯўзҡ„еӯ—з¬ҰдёІ
    3. з¬¬дёүдёӘдҪҚзҪ®пјҡиЎЁзӨәд»ҺеҪ“еүҚзЈҒеӨҙжүҖеңЁдҪҚзҪ®зҡ„дёӢдёҖдёӘдҪҚзҪ®ејҖе§ӢпјҢзӣҙеҲ°жңҖеҗҺдёҖдёӘйқһз©әзҷҪз¬ҰеҸ·дёәжӯўзҡ„еӯ—з¬ҰдёІ

    дёҠеӣҫдёӯдёүдёӘеӣҫзҒөжңәеҪ“еүҚзҡ„й…ҚзҪ®еҰӮдёӢ

    1. \((q, \rhd a, aba)\)
    2. \((h, \rhd\sqcup\sqcup\sqcup, \sqcup a)\)
    3. \((q, \rhd \sqcup a \sqcup\sqcup, e)\)

    д№ҹеҸҜд»Ҙиҝҷж ·иЎЁзӨәй…ҚзҪ®пјҡ\((q, \rhd \underline{a}aba)\), \((h, \rhd\sqcup\sqcup \underline{\sqcup}\sqcup a)\), \((q, \rhd \sqcup a \sqcup \underline{\sqcup})\)пјҢдёӢеҲ’зәҝиЎЁзӨәеҪ“еүҚзЈҒеӨҙжүҖеңЁзҡ„дҪҚзҪ®

    иҖғиҷ‘дёӨдёӘй…ҚзҪ® \((q_1, w_1\underline{a_1}u_1)\), \((q_2, w_2\underline{a_2}u_2)\)пјҢйӮЈд№Ҳ \((q_1, w_1\underline{a_1}u_1) \vdash_M (q_2, w_2\underline{a_2}u_2)\) еҪ“дё”д»…еҪ“еҜ№дәҺжҹҗдёӘ \(b \in \Sigma \cup \lbrace\leftarrow,\rightarrow\rbrace\)пјҢжңү \(\delta(q_1, a_1) = (q_2, b)\)пјҢе№¶дё”ж»Ўи¶ідёӢеҲ—жғ…еҶөд№ӢдёҖпјҡ

    1. \(b \in \Sigma, w_1 = w_2, u_1 = u_2, a_2 = b\)

    2. \(b = \leftarrow, w_1 = w_2a_2\)пјҢе№¶дё”

      1. \(u_2 = a_1u_1\)пјҢеҰӮжһң \(a_1 \not ={\sqcup}\) жҲ– \(u_1 \not ={e}\)пјҢжҲ–иҖ…
      2. \(u_2 = e\)пјҢеҰӮжһң \(a_1 = \sqcup\) дё” \(u_1 = e\)

    3. \(b = \rightarrow, w_2 = w_1a_1\)пјҢе№¶дё”

      1. \(u_1 = a_2u_2\)пјҢжҲ–иҖ…
      2. \(u_1 = u_2 = e\) дё” \(a_2 = \sqcup\)

    д»Ө \(\vdash_M^*\) жҳҜ \(\vdash_M\) зҡ„ reflexive, transitive й—ӯеҢ…

    жҲ‘д»¬иҜҙй…ҚзҪ® \(C_1\) yield й…ҚзҪ® \(C_2\) еҰӮжһң \(C_1 \vdash_M^* C_2\)

    дёҖдёӘ computation жҳҜдёҖдәӣй…ҚзҪ®зҡ„еәҸеҲ— \(C_0 \vdash_M C_1 \vdash_M \cdots \vdash_M C_n\)гҖӮз§°иҝҷдёӘи®Ўз®— length дёә \(n\) жҲ–жңү \(n\) stepsпјҢеҶҷдҪң \(C_0 \vdash_M^n C_n\)

    1.1 A Notation for Turing Machines¶

    е®ҡд№үдёҖдёӘйқһеёёз®ҖеҚ•зҡ„ basic machines еә“пјҢд»ҘеҸҠз»„еҗҲжңәеҷЁзҡ„и§„еҲҷ

    symbol-writing machines and head-moving machinesпјҡеӣәе®ҡжңәеҷЁзҡ„еӯ—жҜҚиЎЁ \(ОЈ\)гҖӮеҜ№дәҺжҜҸдёӘ \(a вҲҲ ОЈ вҲӘ \lbrace вҶ’, вҶҗ \rbrace вҲ’ \lbrace\rhd\rbrace\)пјҢе®ҡд№үдёҖеҸ°еӣҫзҒөжңә \(M_a = (\lbrace s, h \rbrace, ОЈ, Оҙ, s, \lbrace h \rbrace)\)пјҢе…¶дёӯеҜ№жҜҸдёӘ \(b вҲҲ ОЈ вҲ’ \lbrace \rhd\rbrace\)пјҢ\(Оҙ(s,b) = (h,a)\)гҖӮиҮӘз„¶ең°пјҢ\(Оҙ(s,\rhd)\) д»Қз„¶жҳҜ \((s, вҶ’)\)гҖӮд№ҹе°ұжҳҜиҜҙпјҢиҝҷеҸ°жңәеҷЁзҡ„дҪңз”ЁеҸӘжҳҜжү§иЎҢеҠЁдҪң \(a\)пјҡеҰӮжһң \(a вҲҲ ОЈ\)пјҢеҲҷеҶҷе…Ҙз¬ҰеҸ· \(a\)пјӣеҰӮжһң \(a вҲҲ \lbraceвҶҗ, вҶ’\rbrace\)пјҢеҲҷжҢү \(a\) жҢҮзӨәзҡ„ж–№еҗ‘з§»еҠЁпјҢз„¶еҗҺз«ӢеҚіеҒңжӯўгҖӮеҪ“з„¶пјҢиҝҷдёӘиЎҢдёәжңүдёҖдёӘдҫӢеӨ–жғ…еҶөпјҡеҰӮжһңжү«жҸҸеҲ°зҡ„з¬ҰеҸ·жҳҜ \(\rhd\)пјҢйӮЈд№ҲжңәеҷЁдјҡеҗ‘еҸіз§»еҠЁ

    з”ұдәҺз¬ҰеҸ·еҶҷе…ҘжңәеҷЁиў«йў‘з№ҒдҪҝз”ЁпјҢжҲ‘д»¬з®ҖеҢ–е…¶еҗҚз§°пјҢзӣҙжҺҘз”Ё \(a\) д»Јжӣҝ \(M_a\)гҖӮд№ҹе°ұжҳҜиҜҙпјҢиӢҘ \(a вҲҲ ОЈ\)пјҢеҲҷ a-writing machine з®Җи®°дёә \(a\)гҖӮеӨҙз§»еҠЁжңәеҷЁ \(M_вҶҗ\) е’Ң \(M_вҶ’\) е°ҶеҲҶеҲ«з®Җи®°дёә \(L\) е’Ң \(R\)

    еӣҫзҒөжңәе°Ҷд»ҘдёҖз§Қзұ»дјјдәҺжңүйҷҗиҮӘеҠЁжңәз»“жһ„зҡ„ж–№ејҸиҝӣиЎҢз»„еҗҲгҖӮеҚ•дёӘжңәеҷЁе°ұеғҸжңүйҷҗиҮӘеҠЁжңәзҡ„зҠ¶жҖҒпјҢжңәеҷЁд№Ӣй—ҙеҸҜд»ҘеғҸжңүйҷҗиҮӘеҠЁжңәзҡ„зҠ¶жҖҒд№Ӣй—ҙйӮЈж ·зӣёдә’иҝһжҺҘгҖӮз„¶иҖҢпјҢд»ҺдёҖеҸ°жңәеҷЁеҲ°еҸҰдёҖеҸ°жңәеҷЁзҡ„иҝһжҺҘдёҚдјҡеңЁз¬¬дёҖеҸ°жңәеҷЁеҒңжӯўд№ӢеүҚжү§иЎҢпјӣеҸӘжңүеҪ“з¬¬дёҖеҸ°жңәеҷЁеҒңжӯўеҗҺпјҢз¬¬дәҢеҸ°жңәеҷЁжүҚдјҡд»Һе…¶еҲқе§ӢзҠ¶жҖҒејҖе§ӢиҝҗиЎҢпјҢжӯӨж—¶зЈҒеёҰе’ҢиҜ»еҶҷеӨҙзҡ„дҪҚзҪ®з”ұз¬¬дёҖеҸ°жңәеҷЁз•ҷдёӢзҡ„зҠ¶жҖҒеҶіе®ҡ

    еӣ жӯӨпјҢеҰӮжһң \(MвӮҒ\)гҖҒ\(MвӮӮ\) е’Ң \(MвӮғ\) жҳҜеӣҫзҒөжңәпјҢеҲҷдёӢеӣҫжүҖзӨәзҡ„жңәеҷЁжҢүеҰӮдёӢж–№ејҸе·ҘдҪңпјҡд»Һ \(MвӮҒ\) зҡ„еҲқе§ӢзҠ¶жҖҒејҖе§ӢпјӣжҢүз…§ \(MвӮҒ\) зҡ„ж–№ејҸиҝҗиЎҢпјҢзӣҙеҲ° \(MвӮҒ\) еҒңжӯўпјӣз„¶еҗҺпјҢеҰӮжһңеҪ“еүҚжү«жҸҸзҡ„з¬ҰеҸ·жҳҜ \(a\)пјҢеҲҷеҗҜеҠЁ \(MвӮӮ\) е№¶жҢүз…§ \(MвӮӮ\) зҡ„ж–№ејҸиҝҗиЎҢпјӣеҗҰеҲҷпјҢеҰӮжһңеҪ“еүҚжү«жҸҸзҡ„з¬ҰеҸ·жҳҜ \(b\)пјҢеҲҷеҗҜеҠЁ \(MвӮғ\) е№¶жҢүз…§ \(MвӮғ\) зҡ„ж–№ејҸиҝҗиЎҢ

    еҒҮи®ҫдёҠеӣҫдёүдёӘеӣҫзҒөжңә \(MвӮҒ\)гҖҒ\(MвӮӮ\) е’Ң \(MвӮғ\) еҲҶеҲ«дёәпјҡ\(MвӮҒ = (KвӮҒ, ОЈ, ОҙвӮҒ, sвӮҒ, HвӮҒ)\)пјҢ\(MвӮӮ = (KвӮӮ, ОЈ, ОҙвӮӮ, sвӮӮ, HвӮӮ)\)пјҢ\(MвӮғ = (KвӮғ, ОЈ, ОҙвӮғ, sвӮғ, HвӮғ)\)пјҢйӮЈд№ҲдёҠеӣҫжүҖзӨәзҡ„з»„еҗҲжңәеҷЁе°ұжҳҜ \(M = (K, ОЈ, Оҙ, s, H)\)

    1. \(K = KвӮҒ вҲӘ KвӮӮ вҲӘ KвӮғ\)
    2. \(s = sвӮҒ\)
    3. \(H = HвӮӮ вҲӘ HвӮғ\)

    еҜ№дәҺжҜҸдёӘ \(Пғ вҲҲ ОЈ\) е’Ң \(q вҲҲ K вҲ’ H\)пјҢ\(Оҙ(q, Пғ)\) зҡ„е®ҡд№үеҰӮдёӢпјҡ

    1. еҰӮжһң \(q вҲҲ KвӮҒ вҲ’ HвӮҒ\)пјҢеҲҷ \(Оҙ(q, Пғ) = ОҙвӮҒ(q, Пғ)\)
    2. еҰӮжһң \(q вҲҲ KвӮӮ вҲ’ HвӮӮ\)пјҢеҲҷ \(Оҙ(q, Пғ) = ОҙвӮӮ(q, Пғ)\)
    3. еҰӮжһң \(q вҲҲ KвӮғ вҲ’ HвӮғ\)пјҢеҲҷ \(Оҙ(q, Пғ) = ОҙвӮғ(q, Пғ)\)

    4. еҰӮжһң \(q вҲҲ HвӮҒ\)

      1. иӢҘ \(Пғ = a\)пјҢеҲҷ \(Оҙ(q, Пғ) = sвӮӮ\)
      2. иӢҘ \(Пғ = b\)пјҢеҲҷ \(Оҙ(q, Пғ) = sвӮғ\)
      3. еҗҰеҲҷ \(Оҙ(q, Пғ) вҲҲ H\)пјҲжІЎеҫ—еҸҳдәҶе°ұеҒңжӯўжңәеҷЁпјү

    е®һйҷ…дёҠе°ұжҳҜд»Ҙж•°еӯҰеҪўејҸжҸҸиҝ°дәҶзҠ¶жҖҒиҪ¬жҚў

    copying machine \(C\)пјҡе°Ҷ \(\sqcup w \underline{\sqcup}\) еҸҳжҲҗ \(\sqcup w \sqcup w \underline{\sqcup}\)пјҲ\(w\) дёӯдёҚиғҪжңү \(\sqcup\) еӯ—з¬Ұпјү

    right-shifting machine \(S_\rightarrow\)пјҡе°Ҷеӯ—з¬ҰдёІ \(\sqcup w \underline{\sqcup}\) еҸҳжҲҗ \(\sqcup \sqcup w \underline{\sqcup}\)пјҲ\(w\) дёӯдёҚиғҪжңү \(\sqcup\) еӯ—з¬Ұпјү

    еҗҢзҗҶжңү left-shifting machine

    2 Computing with Turing Machines¶

    жҲ‘д»¬йҮҮз”Ёд»ҘдёӢзӯ–з•ҘжқҘеҗ‘еӣҫзҒөжңәжҸҗдҫӣиҫ“е…Ҙпјҡиҫ“е…Ҙеӯ—з¬ҰдёІпјҲдёҚеҗ«з©әзҷҪз¬ҰеҸ·пјүеҶҷеңЁжңҖе·Ұдҫ§з¬ҰеҸ· \(\rhd\) зҡ„еҸідҫ§пјҢе…¶е·Ұдҫ§жңүдёҖдёӘз©әзҷҪпјҢеҸідҫ§д№ҹжңүз©әзҷҪпјӣиҜ»еҶҷеӨҙдҪҚдәҺ \(\rhd\) дёҺиҫ“е…Ҙд№Ӣй—ҙзҡ„з©әзҷҪж–№ж јдёҠпјӣжңәеҷЁд»ҺеҲқе§ӢзҠ¶жҖҒејҖе§ӢиҝҗиЎҢгҖӮеҰӮжһң \(M = (K, \Sigma, \delta, s, H)\) жҳҜдёҖеҸ°еӣҫзҒөжңәпјҢдё” \(w \in (\Sigma - \lbrace \sqcup, \rhd\rbrace)^*\)пјҢйӮЈд№Ҳ \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠзҡ„ initial configuration жҳҜ \((s, \rhd \underline{\sqcup}w)\)

    и®ҫ \(M=(K,ОЈ,Оҙ,s,H)\) жҳҜдёҖеҸ°еӣҫзҒөжңәпјҢе…¶дёӯеҒңжңәзҠ¶жҖҒйӣҶеҗҲ \(H=\lbrace y,n \rbrace\) еҢ…еҗ«дёӨдёӘдёҚеҗҢзҡ„еҒңжңәзҠ¶жҖҒпјҲ\(y\) иЎЁзӨәжҳҜпјҢ\(n\) иЎЁзӨәеҗҰпјүгҖӮд»»дҪ•еҒңжңәй…ҚзҪ®пјҢиӢҘе…¶зҠ¶жҖҒеҲҶйҮҸдёә \(y\)пјҢз§°дёәдёҖдёӘ accepting configurationпјӣиӢҘе…¶зҠ¶жҖҒеҲҶйҮҸдёә \(n\)пјҢеҲҷз§°дёәдёҖдёӘ rejecting configurationгҖӮжҲ‘д»¬иҜҙ \(M\) accepting иҫ“е…Ҙ \(wвҲҲ(ОЈвҲ’\lbraceвҠ”,в–№\rbrace)^вҲ—\)пјҢеҪ“дё”д»…еҪ“ \((s,в–№\underline{вҠ”}w)\) еҜјиҮҙдёҖдёӘжҺҘеҸ—й…ҚзҪ®пјӣзұ»дјјең°пјҢжҲ‘д»¬иҜҙ \(M\) reject иҫ“е…Ҙ \(w\)пјҢеҪ“дё”д»…еҪ“ \((s,в–№\underline{вҠ”}w)\) еҜјиҮҙдёҖдёӘжӢ’з»қй…ҚзҪ®

    и®ҫ \(\Sigma_0 \in \Sigma - \lbrace \sqcup, \rhd\rbrace\) дёәдёҖдёӘеӯ—жҜҚиЎЁпјҢз§°дёә \(M\) зҡ„ input alphabetпјӣйҖҡиҝҮе°Ҷ \(ОЈ_0\) еӣәе®ҡдёә \(ОЈвҲ’\lbraceвҠ”,в–№\rbrace\) зҡ„дёҖдёӘеӯҗйӣҶпјҢжҲ‘д»¬е…Ғи®ёеӣҫзҒөжңәеңЁи®Ўз®—иҝҮзЁӢдёӯдҪҝз”ЁйҷӨиҫ“е…ҘдёӯеҮәзҺ°зҡ„з¬ҰеҸ·д№ӢеӨ–зҡ„йўқеӨ–з¬ҰеҸ·гҖӮжҲ‘д»¬иҜҙ \(M\) decide дёҖдёӘиҜӯиЁҖ \(LвҠҶОЈ_0^*\)пјҢеҪ“дё”д»…еҪ“еҜ№дәҺд»»ж„Ҹеӯ—з¬ҰдёІ \(w \in \Sigma_0^*\)пјҢж»Ўи¶ід»ҘдёӢжқЎд»¶пјҡеҰӮжһң \(wвҲҲL\)пјҢеҲҷ \(M\) жҺҘеҸ— \(w\)пјӣеҰӮжһң \(w\notin L\)пјҢеҲҷ \(M\) жӢ’з»қ \(w\)

    з§°дёҖдёӘиҜӯиЁҖ L дёә recursiveпјҲйҖ’еҪ’зҡ„пјүпјҢеҰӮжһңеӯҳеңЁдёҖеҸ°еӣҫзҒөжңәиғҪеҲӨе®ҡе®ғ

    жҚўеҸҘиҜқиҜҙпјҢдёҖеҸ°еӣҫзҒөжңәеҲӨе®ҡдёҖдёӘиҜӯиЁҖ \(L\)пјҢж„Ҹе‘ізқҖеҪ“д»Ҙиҫ“е…Ҙ \(w\) еҗҜеҠЁж—¶пјҢе®ғжҖ»жҳҜдјҡеҒңжӯўпјҢе№¶дё”еңЁжӯЈзЎ®зҡ„еҒңжңәзҠ¶жҖҒдёӯеҒңжӯўпјҡеҰӮжһңжҳҜ \(y\)пјҢеҲҷ \(wвҲҲL\)пјӣеҰӮжһңжҳҜ \(n\)пјҢеҲҷ \(w\notin L\)гҖӮиҜ·жіЁж„ҸпјҢеҰӮжһңиҫ“е…ҘеҢ…еҗ«з©әзҷҪжҲ–е·Ұз«Ҝз¬ҰеҸ·пјҢеҲҷдёҚдҝқиҜҒдјҡеҸ‘з”ҹд»Җд№Ҳ

    2.1 Recursive Functions¶

    и®ҫ \(M=(K,ОЈ,Оҙ,s,\lbrace h\rbrace)\) жҳҜдёҖеҸ°еӣҫзҒөжңәпјҢд»Ө \(ОЈ_0вҠҶОЈвҲ’\lbraceвҠ”,в–№\rbrace\) дёәдёҖдёӘеӯ—жҜҚиЎЁпјҢдё” \(wвҲҲОЈ_0^вҲ—\)вҖӢгҖӮеҒҮи®ҫ \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠеҒңжӯўпјҢе№¶дё”еӯҳеңЁжҹҗдёӘ \(yвҲҲОЈ_0^вҲ—\)пјҢдҪҝеҫ—д»ҺеҲқе§Ӣй…ҚзҪ® \((s,в–№\underline{вҠ”}w)\) еҮәеҸ‘пјҢз»ҸиҝҮиӢҘе№ІжӯҘеҗҺиҝӣе…ҘеҒңжңәй…ҚзҪ® \((h,в–№\underline{вҠ”}y)\)гҖӮйӮЈд№Ҳз§° \(y\) дёә \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠзҡ„иҫ“еҮәпјҢи®°дҪң \(M(w)\)гҖӮжіЁж„ҸпјҢ\(M(w)\) еҸӘжңүеңЁ \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠеҒңжӯўж—¶жүҚиў«е®ҡд№үпјҢиҖҢдё”е®һйҷ…дёҠеҝ…йЎ»жҳҜеңЁеҪўеҰӮ \((h,в–№\underline{вҠ”}y)\) зҡ„й…ҚзҪ®дёӯеҒңжӯўпјҢе…¶дёӯ \(yвҲҲОЈ_0^вҲ—\)

    зҺ°еңЁи®ҫ \(f\) жҳҜд»Һ \(\Sigma_0^*\) еҲ° \(\Sigma_0^*\) зҡ„д»»ж„ҸеҮҪж•°гҖӮжҲ‘д»¬иҜҙеӣҫзҒөжңә \(M\) compute еҮҪж•° \(f\)пјҢеҰӮжһңеҜ№жүҖжңү \(wвҲҲ\Sigma_0^*\)пјҢйғҪжңү \(M(w)=f(w)\)гҖӮд№ҹе°ұжҳҜиҜҙпјҢеҜ№дәҺжүҖжңү \(wвҲҲ\Sigma_0^*\)пјҢеҪ“ \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠиҝҗиЎҢж—¶пјҢжңҖз»ҲдјҡеҒңжӯўпјҢдё”е…¶зЈҒеёҰеҢ…еҗ«еӯ—з¬ҰдёІ \(в–№вҠ”f(w)\)гҖӮиӢҘеӯҳеңЁдёҖеҸ°еӣҫзҒөжңә \(M\) иғҪи®Ўз®—еҮҪж•° \(f\)пјҢеҲҷз§°еҮҪж•° \(f\) дёә recursive

    йӣҶеҗҲ \(\lbrace0,1\rbrace^вҲ—\) дёӯзҡ„еӯ—з¬ҰдёІеҸҜд»Ҙз”ЁзҶҹжӮүзҡ„дәҢиҝӣеҲ¶иЎЁзӨәжі•жқҘиЎЁзӨәйқһиҙҹж•ҙж•°гҖӮд»»ж„Ҹеӯ—з¬ҰдёІ \(w = a_1a_2 \cdots a_n \in \lbrace 0,1 \rbrace^*\) иЎЁзӨәзҡ„ж•°еҖјдёәпјҡ\(num(w) = a_1 \cdot 2^{n-1} + a_2 \cdot 2^{n-2} + \cdots + a_n\)гҖӮе№¶дё”пјҢд»»дҪ•иҮӘз„¶ж•°йғҪеҸҜд»ҘйҖҡиҝҮдёҖдёӘеҪўеҰӮ \(0вҲӘ1(0вҲӘ1)^вҲ—\) зҡ„еӯ—з¬ҰдёІе”ҜдёҖең°иЎЁзӨәпјҢд№ҹе°ұжҳҜиҜҙпјҢдёҚеҢ…еҗ«еүҚеҜјйӣ¶пјҲеҚіејҖеӨҙеӨҡдҪҷзҡ„ \(0\)пјү

    еӣ жӯӨпјҢи®Ўз®—д»Һ \(\lbrace0,1\rbrace^вҲ—\) еҲ° \(\lbrace0,1\rbrace^вҲ—\) зҡ„еҮҪж•°зҡ„еӣҫзҒөжңәпјҢеҸҜд»Ҙиў«зңӢдҪңжҳҜи®Ўз®—д»ҺиҮӘз„¶ж•°еҲ°иҮӘз„¶ж•°зҡ„еҮҪж•°гҖӮдәӢе®һдёҠпјҢе…·жңүеӨҡдёӘеҸӮж•°зҡ„ж•°еҖјеҮҪж•°пјҢдҫӢеҰӮеҠ жі•е’Ңд№ҳжі•пјҢеҸҜд»ҘйҖҡиҝҮеӣҫзҒөжңәжқҘи®Ўз®—пјҢиҝҷдәӣеӣҫзҒөжңәи®Ўз®—зҡ„жҳҜд»Һ \(\lbrace 0,1,; \rbrace^*\) еҲ° \(\lbrace 0,1\rbrace^*\)пјҢе…¶дёӯ \(;\) жҳҜдёҖдёӘз”ЁдәҺеҲҶйҡ”дәҢиҝӣеҲ¶еҸӮж•°зҡ„з¬ҰеҸ·

    и®ҫ \(M=(K,ОЈ,Оҙ,s,\lbrace h\rbrace)\) жҳҜдёҖеҸ°еӣҫзҒөжңәпјҢе…¶дёӯз¬ҰеҸ· \(0,1,;вҲҲОЈ\)пјҢд»Ө \(f\) жҳҜд»Һ \(N^k\) еҲ° \(N\) зҡ„д»»ж„ҸеҮҪж•°пјҲеҜ№жҹҗдёӘ \(kвүҘ1\)пјүгҖӮжҲ‘д»¬иҜҙеӣҫзҒөжңә \(M\) и®Ўз®—еҮҪж•° \(f\)пјҢеҰӮжһңеҜ№дәҺжүҖжңү \(w_1, \cdots, w_k \in 0 \cup 1 (0 \cup 1)^*\)пјҲеҚіпјҢд»»дҪ• \(k\) дёӘж•ҙж•°зҡ„дәҢиҝӣеҲ¶зј–з Ғеӯ—з¬ҰдёІпјүпјҢйғҪжңүпјҡ\(num(M(w_1;\cdots;w_k)) = f(num(w_1), \cdots, num(w_k))\)

    д№ҹе°ұжҳҜиҜҙпјҢеҰӮжһң \(M\) д»Ҙж•ҙж•° \(n_1,\cdots,n_k\) зҡ„дәҢиҝӣеҲ¶иЎЁзӨәдҪңдёәиҫ“е…ҘпјҢеҲҷе®ғжңҖз»ҲдјҡеҒңжӯўпјҢе№¶дё”еҪ“е®ғеҒңжӯўж—¶пјҢе…¶зЈҒеёҰдёҠеҢ…еҗ«дёҖдёӘиЎЁзӨәж•°еҖј \(f(n_1,\cdots,n_k)\) зҡ„еӯ—з¬ҰдёІпјҢеҚіиҜҘеҮҪж•°зҡ„еҖјгҖӮиӢҘеӯҳеңЁдёҖеҸ°еӣҫзҒөжңә \(M\) иғҪи®Ўз®—еҮҪж•° \(f\)пјҢеҲҷз§°еҮҪж•° \(f: N^k \rightarrow N\) дёә recursive

    2.2 Recursively Enumerable Languages¶

    и®ҫ \(M=(K,ОЈ,Оҙ,s,H)\) жҳҜдёҖеҸ°еӣҫзҒөжңәпјҢд»Ө \(ОЈ_0вҠҶОЈвҲ’{вҠ”,в–№}\) дёәдёҖдёӘеӯ—жҜҚиЎЁпјҢе№¶и®ҫ \(LвҠҶОЈ_0^вҲ—\) дёәдёҖз§ҚиҜӯиЁҖгҖӮжҲ‘д»¬иҜҙ \(M\) semidecidesпјҲеҚҠеҲӨе®ҡпјүиҜӯиЁҖ \(L\)пјҢеҪ“дё”д»…еҪ“еҜ№дәҺд»»ж„Ҹеӯ—з¬ҰдёІ \(wвҲҲОЈ_0^вҲ—\)пјҢд»ҘдёӢжқЎд»¶жҲҗз«Ӣпјҡ\(wвҲҲL\) еҪ“дё”д»…еҪ“ \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠеҒңжңә

    дёҖдёӘиҜӯиЁҖ \(L\) жҳҜ recursively enumerableпјҲйҖ’еҪ’еҸҜжһҡдёҫпјүпјҢеҪ“дё”д»…еҪ“еӯҳеңЁдёҖеҸ°еӣҫзҒөжңә \(M\) иғҪеӨҹеҚҠеҲӨе®ҡ \(L\)

    еӣ жӯӨпјҢеҪ“ \(M\) жҺҘж”¶еҲ°иҫ“е…Ҙ \(wвҲҲL\) ж—¶пјҢе®ғеҝ…йЎ»жңҖз»ҲеҒңжңәгҖӮжҲ‘д»¬е№¶дёҚе…іеҝғе®ғе…·дҪ“иҝӣе…Ҙе“ӘдёӘеҒңжңәзҠ¶жҖҒпјҢеҸӘиҰҒе®ғжңҖз»ҲиҫҫеҲ°жҹҗдёӘеҒңжңәй…ҚзҪ®еҚіеҸҜгҖӮз„¶иҖҢпјҢеҰӮжһң \(wвҲҲОЈ_0^вҲ— вҲ’ L\)пјҢеҲҷ \(M\) ж°ёиҝңдёҚиғҪиҝӣе…ҘеҒңжңәзҠ¶жҖҒ

    жҲ‘д»¬и®° \(M(w)=вҶ—\) иЎЁзӨә \(M\) еңЁиҫ“е…Ҙ \(w\) дёҠж— жі•еҒңжңәгҖӮеңЁжӯӨи®°жі•дёӢпјҢжҲ‘д»¬еҸҜд»Ҙе°ҶеӣҫзҒөжңә \(M\) еҚҠеҲӨе®ҡиҜӯиЁҖ \(LвҠҶОЈ_0^вҲ—\) зҡ„е®ҡд№үйҮҚиҝ°еҰӮдёӢпјҡеҜ№жүҖжңү \(wвҲҲОЈ_0^вҲ—\)пјҢжңү \(M(w)=вҶ—\) еҪ“дё”д»…еҪ“ \(w\notin L\)

    еҰӮжһңдёҖдёӘиҜӯиЁҖжҳҜ recursive зҡ„пјҢйӮЈд№Ҳе®ғд№ҹжҳҜ recursively enumerable зҡ„

    еӯҳеңЁ recursively enumerable дҪҶжҳҜдёҚ recursive зҡ„иҜӯиЁҖ

    иҜҒжҳҺ

    иҰҒжһ„йҖ дёҖеҸ°еҚҠеҲӨе®ҡпјҲиҖҢдёҚжҳҜеҲӨе®ҡпјүиҜҘиҜӯиЁҖзҡ„еӣҫзҒөжңәпјҢеҸӘйңҖе°ҶеҺҹеҲӨе®ҡжңәдёӯзҡ„жӢ’з»қзҠ¶жҖҒеҸҳдёәйқһеҒңжңәзҠ¶жҖҒпјҢд»ҺиҖҢзЎ®дҝқжңәеҷЁж°ёиҝңдёҚдјҡеҒңжңәгҖӮе…·дҪ“жқҘиҜҙпјҢз»ҷе®ҡд»»ж„ҸдёҖеҸ°еҲӨе®ҡиҜӯиЁҖ \(L\) зҡ„еӣҫзҒөжңә \(M=(K,ОЈ,Оҙ,s,\lbrace y,n \rbrace)\)пјҢжҲ‘д»¬еҸҜд»Ҙе®ҡд№үдёҖеҸ°еҚҠеҲӨе®ҡ \(L\) зҡ„ж–°жңәеҷЁ \(M'\) еҰӮдёӢпјҡ\(M' = (K,ОЈ,\delta',s,\lbrace y\rbrace)\)пјҢе…¶дёӯ \(\delta'\) жҳҜ \(Оҙ\) зҡ„жү©еұ•пјҢеўһеҠ еҰӮдёӢиҪ¬з§»и§„еҲҷпјҡеҜ№жүҖжңү \(aвҲҲОЈ\)пјҢжңү \(\delta'(n,a) = (n,a)\) вҖ”вҖ” еҚізҠ¶жҖҒ \(n\) дёҚеҶҚжҳҜеҒңжңәзҠ¶жҖҒгҖӮжҳҫз„¶пјҢеҰӮжһң \(M\) зЎ®е®һиғҪеҲӨе®ҡ \(L\)пјҢйӮЈд№Ҳ \(M'\) е°ұиғҪеҚҠеҲӨе®ҡ \(L\)пјҢеӣ дёә \(M'\) жҺҘеҸ—дёҺ \(M\) зӣёеҗҢзҡ„иҫ“е…ҘпјӣжӯӨеӨ–пјҢеҰӮжһң \(M\) жӢ’з»қиҫ“е…Ҙ \(w\)пјҢеҲҷ \(M'\) еңЁ \(w\) дёҠдёҚдјҡеҒңжңәпјҲе®ғеңЁзҠ¶жҖҒ \(n\) дёӯж°ёиҝңеҫӘзҺҜпјүгҖӮжҚўеҸҘиҜқиҜҙпјҢеҜ№дәҺжүҖжңүиҫ“е…Ҙ \(w\)пјҢжңү \(M'(w)=вҶ—\) еҪ“дё”д»…еҪ“ \(M(w)=n\)

    еҰӮжһңиҜӯиЁҖ \(L\) жҳҜ recursive зҡ„пјҢйӮЈд№Ҳ \(\bar{L}\) д№ҹжҳҜ recursive зҡ„

    3 Extensions of the Turing Machine¶

    3.1 Multiple Tapes¶

    еҸҜд»Ҙи®ҫжғіе…·жңүеӨҡдёӘзЈҒеёҰзҡ„еӣҫзҒөжңәпјҢжҜҸжқЎзЈҒеёҰйҖҡиҝҮдёҖдёӘиҜ» / еҶҷеӨҙдёҺжңүйҷҗжҺ§еҲ¶еҷЁзӣёиҝһпјҲжҜҸжқЎзЈҒеёҰжңүдёҖдёӘиҜ» / еҶҷеӨҙпјүгҖӮеңЁдёҖжӯҘж“ҚдҪңдёӯпјҢжңәеҷЁеҸҜд»ҘиҜ»еҸ–жүҖжңүиҜ» / еҶҷеӨҙжүҖжү«жҸҸзҡ„з¬ҰеҸ·пјҢз„¶еҗҺж №жҚ®иҝҷдәӣз¬ҰеҸ·еҸҠе…¶еҪ“еүҚзҠ¶жҖҒпјҢйҮҚеҶҷжҹҗдәӣиў«жү«жҸҸзҡ„ж–№ж јпјҢе№¶е°ҶжҹҗдәӣиҜ» / еҶҷеӨҙеҗ‘е·ҰжҲ–еҗ‘еҸіз§»еҠЁпјҢеҗҢж—¶ж”№еҸҳиҮӘиә«зҠ¶жҖҒ
